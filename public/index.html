API Usage

Base URL: https://image-store-api-sigma.vercel.app

GET /property/random

Get a random property image

Example:

https://image-store-api-sigma.vercel.app/property/random?q=80
GET /property/:id

Get a specific property image by ID

Example:

https://image-store-api-sigma.vercel.app/property/2?q=80

Deploy Your Own Instance

Want to run your own instance? Deploy to Vercel with one click!

Deploy to Vercel

Or deploy manually:

  1. Fork the repository
  2. Clone your fork
  3. Deploy using Vercel CLI or connect your repository through Vercel's dashboard
